About Batch Statements

The Batch Statements wizard allows users with the proper permissions to print talent profile and performance evaluation statements for multiple employees in batches. For example, if you have 50 talent profile statements to print, you can use this wizard to select the employees and print their statements in small batches (to reduce loading times). The batch size is configurable using the Bulk Statements Batch Size system setting.  

To get to the Batch Statements wizard, you have the following options:

  • Go to Primary Navigation Menu > Reports > Batch Statements.

  • You can access batch evaluation statements from the Status of Active Processes widget, if you have employees in the Completed state of an evaluation workflow process. In this case, you can click the button next to the Process name, or click the number in the Completed state.

  • Click the Print Statements button in any My Team SmartGrid or in the grid layout of a Calibration Session.

The wizard guides you through the steps necessary for displaying either Talent Profile statements or Performance Evaluation statements in Microsoft Word or Adobe PDF format. You can then print the resulting documents (one file per batch) using that application's print function, or save the files for future reference. To learn how to do this, see Batch Print Statements.

  1. Note: Statements designed for single printing do not format properly for batch printing. You must change the statement design for use with batch printing. For more information about BIRT and other applications required to customize statements, see Report Customizations. Contact your PeopleFluent representative if you have any further questions about changing the design of a statement.

Security

Two security groups control access to batch statements: Can Access Performance Evaluation Batch Statements and Can Access Talent Profile Batch Statements.

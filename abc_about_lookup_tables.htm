About Lookup Tables

PeopleFluent Talent Management supports a highly flexible lookup framework that lets administrators define and reference lookup objects, or tables of data, within formulas. A lookup table consists of a set of data that converts information by matching "raw" input values against a set of pre-defined values in the lookup table and arrives at a discrete result.

Administrators can add and configure as many lookup tables as necessary for their environment. If you have pre-defined lookup tables that you want to reuse, you can easily import sets of lookup tables from one goal definition, performance plan, or reward cycle to another. You can also copy lookup tables within the same goal, plan, or reward cycle.

For more information, see Create Lookup Tables.

  1. Note: Lookup tables should be defined before configuring formulas so that formulas can access lookups when needed.

Lookups are used by (click the text to expand it):

  • Goals

    Goals can be configured to use lookup tables to define more detailed payout rules based on employee performance and ratings. To access lookup tables for goals, go to Primary Navigation Menu > Talent Management Administration > Goal Management > Global Goal Repository > Goal Repository. On the Goal Repository page, click the Edit icon next to a goal, and then click Measurements. On the Goal Measurements page, click Advanced Calculation (in the Incentive Payout Calculations section) to display the Lookup Tables item in the Menu Bar at left.  

  • Performance Management Plans

    In performance management plans, lookup tables can be used in conjunction with formulas to customize the entire performance process. For example, in performance plans that use automatically calculated goal ratings, lookup tables can be used to map specific performance targets to rating values. To access lookup tables for performance, go to Primary Navigation Menu > Talent Management Administration > Performance Administration > Performance Management > Performance Management Plans. On the Performance Management Plans page, click the Edit icon next to a plan, and then click Lookup Tables under Advanced Calculations.

  • Compensation Plans and Reward Cycles

    Lookup tables can be used in compensation formulas to perform a variety of functions. Lookups can return a minimum or maximum range for compensation awards, or return a rounding factor, to provide the same results for a compensation plan that supports several different currencies.

To access lookup tables for compensation plans, you have the following options:

  • For salary plans, go to Primary Navigation Menu > Talent Management Administration > Compensation Administration > Salary Planning > Salary Plan Setup. On the Salary Plan Configurations page, click the Edit icon next to a plan, and then click Lookup Tables.

  • For bonus plans, go to Primary Navigation Menu > Talent Management Administration > Compensation Administration > Bonus Planning > Bonus Plan Setup. On the Bonus Plan Configurations page, click the Edit icon next to a plan, and then click Lookup Tables.

  • For stock plans, go to Primary Navigation Menu > Talent Management Administration > Compensation Administration > Stock Planning > Stock Plan Setup. On the Stock Plan Configurations page, click the Edit icon next to a plan, and then click Lookup Tables.

To access lookup tables for compensation reward cycles, you have the following options:

  • For bonus reward cycles, go to Primary Navigation Menu > Talent Management Administration > Compensation Administration > Bonus Planning > Reward Cycle Configurations. On the Reward Cycle Configurations page, click the Edit icon next to a reward cycle, and then click Lookup Tables.

  • For salary reward cycles, go to Primary Navigation Menu > Talent Management Administration > Compensation Administration > Salary Planning > Reward Cycle Configurations. On the Reward Cycle Configurations page, click the Edit icon next to a reward cycle, and then click Lookup Tables.

  • For stock reward cycles, go to Primary Navigation Menu > Talent Management Administration > Compensation Administration > Stock Planning > Reward Cycle Configurations. On the Reward Cycle Configurations page, click the Edit icon next to a reward cycle, and then click Lookup Tables.

  • For total compensation reward cycles, go to Primary Navigation Menu > Talent Management Administration > Compensation Administration > Total Compensation Planning > Reward Cycle Configurations. On the Reward Cycle Configurations page, click the Edit icon next to a reward cycle, and then click Lookup Tables.

Examples

Lookup tables can be constructed for a variety of purposes, as shown in the examples below (click the example text to expand it):

  • Example 1: Goal Achievement Payout Rules

    This simple lookup table is for calculating the payout for a specific goal achievement. This example shows a single-input lookup table, with a single percentage output titled Payout. The input factor is the percentage of achievement that the employee attained in relation to the specific goal. For example, if the employee achieved 80% of a given goal, the payout would be 50% in the period covered by the goal plan.

    Figure: Goal Achievement Payout Rules

  • Example 2: Calculated Goal Ratings

    This lookup table is for a calculated goal rating, typically used in performance management plans where some goals are automatically rated by the system. This example shows a single-input lookup table, with a single numeric output titled Rating. The input factor is the number of units sold, and it returns a rating from 1 to 5, corresponding to the goal rating scale used in the plan.

    Figure: Calculated Goal Ratings

  • Example 3: Precision Rounding Rules

    Another example would be a lookup table that returns the precision/rounding factor, for a compensation plan that supports several different currencies. Once defined, the lookup table is referenced in a formula to use the return value "precision" when calling the rounding function. In this example, a single-input lookup table would be created, with a single numeric output, titled Precision.

    Figure: Precision Rounding Rules Lookup Table

    Lookup tables can also be designed to return a list of values for compensation awards, such as salary increases. The following lookup table uses both a horizontal axis and a vertical axis, and returns a single output, in this case, a list of percentages (titled Min and Max), based on an employee's compa-ratio and performance rating, that represents an allowable range for salary increase.
